# Required Dependencies for Health Metrics Feature

## Install Recharts for Charts

Run this command to install the charting library:

```bash
npm install recharts
```

Or with yarn:

```bash
yarn add recharts
```

## What is Recharts?

Recharts is a composable charting library built on React components. It's used for:
- Line charts (Blood Pressure, Blood Sugar, Weight trends)
- Bar charts (Medication adherence)
- Area charts (Health metrics over time)
- Responsive and interactive visualizations

## After Installation

Once installed, restart the development server:

```bash
npm run dev
```

The Health Metrics Dashboard will then display interactive charts for all health data!
